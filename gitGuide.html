git 튜토리얼

  1. git 다운로드 https://git-scm.com/downloads 이후 git bash 실행
  2. git config --global user.name "이름"
  3. git config --global user.email "이메일"
    • git config --list : 이름과 이메일이 정상적으로 입력됐는지 확인.
  4. git init : 프로젝트를 맨처음 올릴때 입력
  5. git add 파일명 : 해당 파일을 git에 추가
    • git status : 현재 깃의 상태를 출력한다.

      초록색 : git add 명령어로 git에 commit할 파일들을 출력한다.(tracked 상태)
      빨간색 :
      gitGuide.html : git add 입력 후 수정된 파일이다. git add를 한적이 있으므로 tracked 상태이다.
      img/gitStatus.jpg : git add 입력 후 새로 만들어진 파일이다.(untracked 상태)
